# GIS Demo

範例使用 Live Server 即可 Demo。

``` bash
├── app/
│   ├── assets/         # 資源
│   │   ├── images/     # 圖片
│   │   ├── map/        # 地圖相關資源
│   │   │   ├── source/ # 地圖來源
│   │   │   └── style/  # 地圖樣式
│   │   └── models/     # 模型
│   ├── libs/           # 涵式庫
│   ├── index.html      # 主頁
│   └── main.js         # 主程式
└── scripts/
    ├── libs/       # 涵式庫
    ├── resources/  # 資源
    └── transfer.js # 將資源轉換成 GeoJSON（npm run transfer）
```
